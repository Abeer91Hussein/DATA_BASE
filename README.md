# Saleh Khalaf Shop Management System

This project is a database-driven management system developed for a famous Palestinian spice shop (Salah Khalaf).
It is designed to manage daily operations such as customers, employees, orders, and inventory in an efficient and organized way.

The system provides an interactive interface for different user roles including customers, employees, and managers.

## Technologies Used

  Python (Flask Framework)
  SQL (Database Design & Queries)
  HTML / CSS (Frontend Templates)


## Features

* Customer account management (signup, login, cart, invoice)
* Employee and manager dashboards
* Product and inventory management
* Order and purchase tracking
* Warehouse and branch management
* Role-based access control

## System Concept

The system simulates a real-world spice shop environment, where:

 Customers can browse products and place orders
 Employees handle daily operations
 Managers oversee reports, staff, and inventory
